# 学生作业管理系统简介 python130

## 项目概述

学生作业管理系统是基于 Python 语言，使用 Django 框架开发的应用。本系统旨在为中小学生、教师及管理员提供一个便捷的作业管理平台，实现作业的发布、提交、批改以及相关信息的集中管理。

## 技术栈

- 开发语言：Python
- 开发框架：Django
- 数据库：MySQL

- 作业批改管理
- 个人信息管理

### 教师角色

## 部署步骤

1. 准备 Python、Django 和 MySQL 环境；
2. 克隆或下载项目代码；
3. 安装依赖包，运行 `pip install -r requirements.txt`；
4. 初始化数据库，运行 `python manage.py makemigrations` 和 `python manage.py migrate`；
5. 运行 `python manage.py runserver`，启动开发服务器。
